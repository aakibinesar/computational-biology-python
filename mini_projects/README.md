# Mini-projects

This folder will contain small applied projects that connect the curriculum to practical computational biology workflows.

The mini-projects are intended to show how individual Rosalind-style problems can be combined into larger, reusable pipelines.

## Planned Mini-projects

| Project | Focus |
|---|---|
| `ml_ready_sequence_features/` | Convert biological sequences into numerical features for baseline ML workflows |

## Main Planned Project

### ML-ready Biological Sequence Features with Python

This mini-project will demonstrate how to:

1. read biological sequences from FASTA-style input;
2. clean and validate sequences;
3. compute GC-content features;
4. extract k-mer frequency features;
5. build a feature matrix using Python, NumPy, and Pandas;
6. apply a simple baseline clustering or classification workflow;
7. interpret biological sequence features.

## Relationship to the Curriculum

The mini-projects connect especially strongly to:

- Module 03: Biological Sequence Basics
- Module 04: Sequence Statistics, Motifs, and k-mers
- Module 06: Dynamic Programming and Sequence Alignment
- Module 12: Practical Bioinformatics with Biopython
