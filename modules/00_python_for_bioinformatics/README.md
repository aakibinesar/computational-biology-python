# Module 00: Python for Bioinformatics

## Purpose

This module introduces the Python programming foundations needed for computational biology.

The goal is to show how basic programming concepts such as variables, strings, loops, files, and dictionaries become practical tools for biological sequence analysis.

This module is based on the Rosalind Python Village problems and prepares learners for later work with DNA, RNA, proteins, FASTA files, k-mers, dynamic programming, graph algorithms, and Biopython workflows.

## Core Concepts

- Python syntax
- Variables and arithmetic
- String slicing
- Loops and conditionals
- File reading
- Dictionaries and counting
- Basic text processing
- Preparing biological sequence data for analysis

## Learning Outcomes

After completing this module, a learner should be able to:

- write simple Python programs;
- manipulate strings using indexing and slicing;
- read input from files;
- use loops and conditionals to process data;
- use dictionaries for counting problems;
- understand how general Python skills transfer to biological sequence analysis.
