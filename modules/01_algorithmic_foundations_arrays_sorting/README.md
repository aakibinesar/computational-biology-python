# Module 01: Algorithmic Foundations — Arrays, Sorting, Searching, and Recurrence

## Purpose

This module introduces core algorithmic techniques that support computational biology.

The focus is on arrays, searching, sorting, recurrence relations, heaps, inversions, selection, and sum-search problems. These concepts appear throughout bioinformatics in sequence processing, k-mer analysis, suffix-based indexing, dynamic programming, genome rearrangements, and feature construction.

## Core Concepts

- Recurrence relations
- Binary search
- Sorting algorithms
- Divide-and-conquer
- Merge sort
- Quick sort
- Heap operations
- Counting inversions
- Majority elements
- Pair and triple sum search
- Selection and partial sorting

## Learning Outcomes

After completing this module, a learner should be able to:

- explain the role of recurrence in algorithm design;
- implement binary search and common sorting methods;
- understand divide-and-conquer strategies;
- use heaps for priority-based computation;
- count inversions as a measure of ordering difference;
- connect classical algorithmic problems to biological data processing.
