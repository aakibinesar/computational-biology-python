# Module 03: Biological Sequence Basics

## Purpose

This module introduces the basic biological sequence operations used throughout computational biology.

The focus is on DNA, RNA, proteins, transcription, reverse complements, translation, open reading frames, splicing, sequence distance, and basic protein properties. These problems form the biological foundation for the later algorithmic and ML-ready parts of the repository.

## Core Concepts

- DNA sequence representation
- RNA transcription
- Reverse complement
- Hamming distance
- Codon translation
- Protein strings
- Open reading frames
- RNA splicing
- Protein mass
- Transition/transversion ratio
- Reverse-palindromic sites
- Subsequence matching

## Learning Outcomes

After completing this module, a learner should be able to:

- represent DNA, RNA, and protein sequences as strings;
- count nucleotides and compute simple sequence properties;
- transcribe DNA into RNA;
- compute reverse complements;
- translate RNA into protein sequences;
- identify candidate open reading frames;
- understand splicing and intron removal;
- compute basic sequence distances and biological sequence properties.
