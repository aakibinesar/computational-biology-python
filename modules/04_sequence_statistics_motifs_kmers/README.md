# Module 04: Sequence Statistics, Motifs, and k-mers

## Purpose

This module focuses on sequence statistics, motif discovery, profile matrices, k-mer counting, and probabilistic sequence analysis.

These ideas are central to bioinformatics because biological sequences often need to be summarized numerically before they can be compared, searched, classified, clustered, or used in machine learning workflows.

## Core Concepts

- GC content
- Motif search
- Consensus sequences
- Profile matrices
- Shared motif discovery
- k-mer composition
- Sequence probability
- Expected motif occurrences
- Error correction in reads
- Approximate pattern matching
- Motif discovery
- Feature-style sequence representation

## Learning Outcomes

After completing this module, a learner should be able to:

- compute GC content and other sequence-level statistics;
- search for motifs in DNA sequences;
- construct consensus sequences and profile matrices;
- identify shared motifs across multiple sequences;
- represent sequences using k-mer counts;
- understand how sequence statistics become machine-learning features;
- connect motif discovery to practical biological sequence analysis.
