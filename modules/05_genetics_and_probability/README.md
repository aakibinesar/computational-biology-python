# Module 05: Genetics and Probability

## Purpose

This module connects combinatorics, probability, and genetics.

The focus is on Mendelian inheritance, dominant phenotype probabilities, allele frequencies, genetic drift, founder effects, binomial expectations, set operations, permutations, and combinatorial counting.

These problems show how mathematical reasoning supports biological modelling.

## Core Concepts

- Mendelian inheritance
- Probability of dominant phenotypes
- Independent allele inheritance
- Allele frequency estimation
- Wright-Fisher genetic drift
- Founder effect
- Binomial expectations
- Combinatorial counting
- Set operations
- Permutations and signed permutations
- Counting biological possibilities

## Learning Outcomes

After completing this module, a learner should be able to:

- compute basic inheritance probabilities;
- model expected phenotype outcomes;
- understand allele-frequency reasoning;
- apply binomial and combinatorial reasoning to biological problems;
- use set and permutation logic in computational biology tasks;
- connect probability-based models to population genetics and biological modelling.
