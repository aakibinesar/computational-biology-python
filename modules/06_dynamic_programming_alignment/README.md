# Module 06: Dynamic Programming and Sequence Alignment

## Purpose

This module introduces dynamic programming as a central method in computational biology.

The focus is on recurrence, sequence comparison, longest common subsequence, edit distance, global alignment, local alignment, affine gap penalties, overlap alignment, semiglobal alignment, and multiple-sequence alignment.

Dynamic programming is one of the most important algorithmic techniques in bioinformatics because many sequence analysis problems can be framed as optimal path or optimal table-filling problems.

## Learning Outcomes

After completing this module, a learner should be able to:

- explain why recurrence leads naturally to dynamic programming;
- construct and fill dynamic programming tables;
- compute longest common subsequences;
- compute edit distance between biological sequences;
- perform global and local sequence alignment;
- understand the role of gap penalties;
- reconstruct alignments using backtracking;
- connect sequence alignment scores to similarity-based biological features.
