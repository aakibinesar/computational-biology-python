# Module 09: RNA Structure and Combinatorial Biology

## Purpose

This module focuses on RNA structure counting and combinatorial biological problems.

RNA structure problems often involve matching bases, counting possible pairings, and using recursive or dynamic programming-style reasoning. This module connects recurrence, combinatorics, and sequence structure to RNA secondary structure analysis.

## Core Concepts

- RNA base pairing
- Perfect matchings
- Maximum matchings
- Catalan-style counting
- Motzkin-style counting
- Wobble base pairs
- Recursive structure counting
- Dynamic programming for RNA structures
- Interwoven motifs
- Combinatorial biological reasoning

## Learning Outcomes

After completing this module, a learner should be able to:

- explain how RNA base-pairing constraints create combinatorial problems;
- count perfect and maximum matchings in RNA sequences;
- understand how Catalan and Motzkin-style recurrences appear in RNA structure;
- reason about wobble base pairs;
- connect recursive counting to dynamic programming;
- interpret RNA structure problems as biological applications of combinatorics.
