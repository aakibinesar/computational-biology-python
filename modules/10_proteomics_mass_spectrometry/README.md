# Module 10: Proteomics and Mass Spectrometry

## Purpose

This module introduces protein and peptide analysis through mass spectrometry-style computational problems.

The focus is on translating RNA into proteins, computing peptide masses, reconstructing peptides from spectra, using spectrum graphs, and scoring peptide-spectrum matches. These problems show how biological sequence analysis extends from DNA/RNA strings to protein-level inference.

## Core Concepts

- RNA translation
- Protein mass calculation
- Peptide spectra
- Cyclic and linear spectra
- Spectral convolution
- Peptide reconstruction
- Spectrum graphs
- Peptide-spectrum matching
- Protein inference
- Mass-based biological sequence analysis

## Learning Outcomes

After completing this module, a learner should be able to:

- translate RNA sequences into proteins;
- compute protein and peptide masses;
- understand linear and cyclic peptide spectra;
- reconstruct peptides from mass information;
- interpret spectrum graphs;
- understand basic peptide-spectrum matching;
- connect proteomics problems to sequence analysis and graph reasoning.
