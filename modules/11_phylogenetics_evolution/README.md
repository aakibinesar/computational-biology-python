# Module 11: Phylogenetics and Evolution

## Purpose

This module introduces computational approaches to evolutionary relationships and phylogenetic trees.

The focus is on distance matrices, Newick tree parsing, weighted tree distances, character tables, quartet reasoning, split distance, and ancestral sequence inference. These problems connect sequence comparison, tree structures, graph reasoning, and evolutionary interpretation.

## Core Concepts

- Pairwise sequence distance
- Distance matrices
- Newick tree format
- Weighted tree distances
- Character tables
- Tree splits
- Quartets
- Split distance
- Phylogenetic reconstruction
- Ancestral sequence inference
- Evolutionary comparison

## Learning Outcomes

After completing this module, a learner should be able to:

- compute pairwise sequence distance matrices;
- understand basic tree representations used in phylogenetics;
- parse and reason about Newick-style tree structures;
- interpret character tables and tree splits;
- understand quartet-based phylogenetic reasoning;
- connect graph/tree algorithms to evolutionary biology;
- relate sequence similarity to evolutionary distance.
