# Module 12: Practical Bioinformatics with Biopython

## Purpose

This module introduces practical bioinformatics workflows using Biopython-style tasks.

The focus is on working with real biological file formats, sequence records, GenBank retrieval, FASTA/FASTQ processing, quality scores, filtering reads, pairwise alignment workflows, translation tables, reverse complements, and ORF detection.

This module connects algorithmic problem solving with practical biological data handling.

## Core Concepts

- Biopython introduction
- Sequence records
- GenBank retrieval
- FASTA handling
- FASTQ handling
- FASTQ to FASTA conversion
- Quality-score interpretation
- Read filtering
- Pairwise alignment workflows
- Translation tables
- Reverse complement counting
- ORF detection workflows

## Learning Outcomes

After completing this module, a learner should be able to:

- understand the role of Biopython in practical sequence analysis;
- work with FASTA and FASTQ-style biological data;
- interpret basic read-quality information;
- filter biological sequence reads;
- retrieve and compare biological sequences;
- connect core sequence algorithms to practical workflows;
- prepare biological sequence data for downstream analysis.
