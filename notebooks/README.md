# Tutorial Notebooks

This folder will contain tutorial-style Jupyter notebooks for the computational biology curriculum.

The notebooks are intended to turn selected Rosalind problems and module concepts into beginner-friendly, executable learning materials.

## Planned Notebooks

| Notebook | Focus |
|---|---|
| `01_python_for_bioinformatics.ipynb` | Python basics for biological sequence processing |
| `02_algorithmic_foundations.ipynb` | Searching, sorting, recurrence, and graph foundations |
| `03_sequence_statistics_motifs_kmers.ipynb` | GC content, motifs, profile matrices, and k-mer features |
| `04_dynamic_programming_alignment.ipynb` | LCS, edit distance, and sequence alignment |
| `05_graphs_genome_assembly.ipynb` | Overlap graphs, de Bruijn graphs, and genome reconstruction |
| `06_biopython_ml_ready_features.ipynb` | FASTA/FASTQ handling and ML-ready sequence features |

## Notebook Goals

The notebooks will focus on:

- clear explanations;
- runnable Python examples;
- biological interpretation;
- reusable helper functions;
- links between algorithms and bioinformatics applications;
- preparation for ML-ready biological sequence analysis.

## Development Status

Notebook development is planned after the repository structure, module READMEs, curriculum map, dependency map, and solved problem index are completed.
