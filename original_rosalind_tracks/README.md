# Original Rosalind Tracks

This folder preserves the original Rosalind solution archive.

The purpose of this folder is to keep the solved problems organized according to their original Rosalind track structure, while the `modules/` folder reorganizes the same work into a teaching-oriented computational biology curriculum.

## Tracks Included

| Folder | Rosalind Track | Purpose |
|---|---|---|
| `python_village/` | Python Village | Basic Python programming foundations |
| `algorithmic_heights/` | Algorithmic Heights | Core algorithms, sorting, searching, graphs, and dynamic programming |
| `bioinformatics_stronghold/` | Bioinformatics Stronghold | Main bioinformatics problem set |
| `bioinformatics_armory/` | Bioinformatics Armory | Practical bioinformatics and Biopython-style workflows |
| `bioinformatics_textbook_track/` | Bioinformatics Textbook Track | Textbook-style computational biology problems |

## Relationship to the Curriculum

The original tracks are preserved here for archival clarity.

The teaching-oriented organization is maintained in:

- `curriculum_map.md`
- `problem_dependency_map.md`
- `solved_problem_index.md`
- `modules/`

## Notes

Files in this folder represent original or lightly reorganized solution material. Expanded explanations, teaching notes, notebooks, and module-level summaries are developed separately in the `modules/`, `notebooks/`, and `teaching/` folders.
