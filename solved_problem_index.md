# Solved Problem Index

This file indexes solved Rosalind problems by track, curriculum module, topic, and teaching role.

The goal is not only to list solved problems, but to show how the original Rosalind tracks connect to a teaching-oriented curriculum in computational biology, algorithms, Biopython workflows, and ML-ready biological sequence analysis.

## Treatment Labels

- **Fully explained** — detailed teaching note, algorithm explanation, code walkthrough, and complexity discussion.
- **Worked example** — clean code with a short explanation.
- **Archive solution** — preserved as a solved problem but not expanded in detail.
- **Cross-module reference** — used in more than one module as a bridge concept.
- **Capstone reference** — used in the ML-ready bioinformatics mini-project.
