# Teaching Materials

This folder will contain teaching-oriented materials derived from the computational biology curriculum.

The goal is to make the repository useful not only as a solution archive, but also as a structured learning resource for students and early-stage researchers interested in Python, algorithms, bioinformatics, and biological sequence analysis.

## Planned Materials

| Folder/File | Purpose |
|---|---|
| `workshop_outline.md` | High-level workshop structure |
| `lesson_plans.md` | Lesson-by-lesson teaching plan |
| `exercises/` | Practice exercises and prompts |
| `slides/` | Future slide materials |

## Teaching Focus

The teaching materials will emphasize:

- Python for biological sequence analysis;
- algorithmic foundations for computational biology;
- string matching and genome indexing;
- graph-based genome assembly;
- dynamic programming and sequence alignment;
- practical Biopython workflows;
- ML-ready biological feature construction.

## Intended Audience

These materials are intended for:

- beginners in computational biology;
- computer science students entering bioinformatics;
- students learning Python through biological examples;
- learners preparing for research in bioinformatics, biomedical AI, or biological data science.
